# License and Notice Policy

Every package must include Sloppy's current license file and notice policy.
The current project license is pending, so artifacts are limited to internal
dry-run validation.

A complete third-party license review is required before publishing.
That review must inventory bundled native runtime libraries, optional V8
runtime files, compiler dependencies, and generated notices required by
upstream dependencies.

Package scripts must not include source-only dependency caches, V8 SDK headers,
V8 import libraries, vcpkg source trees, or build directories.
