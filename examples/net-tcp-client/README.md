# TCP Client Example

This source example shows the scoped `sloppy/net` TCP client API shape for a
loopback service: `NetworkAddress.parse`, `TcpClient.connect`, `writeText`,
`readLine`, deadline propagation, `noDelay`, `keepAlive`, and deterministic close.

It is not a TLS client, HTTP client, UDP client, WebSocket client, Node/Bun/Deno
compatibility claim, package-manager behavior, public release documentation,
benchmark evidence, or external live-network evidence.
