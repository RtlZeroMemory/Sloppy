# Framework v2 Explicit Binding Example

This is a compile-time Framework v2 binding example.
This example exercises explicit route, query, header, body, and request-context handler
bindings. It demonstrates deterministic binding metadata for the current
supported wrapper subset.

## Limitations

The example is limited to the explicit binding wrappers shown in `app.ts`.
Decorators, service scanning, and broad coercion behavior are outside this
example.
