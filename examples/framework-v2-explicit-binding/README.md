# Framework v2 Explicit Binding Example

Status: compile-time Framework v2 binding example.

This example exercises explicit route, query, header, body, and request-context handler
bindings. It proves the compiler can emit deterministic binding metadata for the current
supported wrapper subset.

This example is source-input/tooling evidence only. It does not claim broader coercion
breadth, decorators, service scanning, Node/Bun/Deno compatibility, production-readiness,
or package-manager behavior.
