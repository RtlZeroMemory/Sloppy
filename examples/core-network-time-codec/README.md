# Core Network Time Codec Example

This example uses LocalEndpoint
operation options, a shared Deadline, and Codec UTF-8 helpers for byte ownership at the
network boundary.

The example documents implemented API shape only. It has no public release claim, no
package-manager behavior, no Node/Bun/Deno compatibility promise, no production-readiness
claim, and no benchmark or performance claim. It does not claim an OS sandbox.
