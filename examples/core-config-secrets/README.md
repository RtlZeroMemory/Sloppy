# Core Config Secrets Example

This example shows Config secret
values moving into response metadata only through the shared redacted string shape.

The example documents implemented API shape only. It has no public release claim, no
package-manager behavior, no Node/Bun/Deno compatibility promise, no production-readiness
claim, and no benchmark or performance claim. Placeholder configuration values must remain
placeholder-only; real tokens do not belong in examples or goldens.
