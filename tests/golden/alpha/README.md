# Alpha Golden Proof Suite

This directory contains normalized golden outputs generated by
`tools/golden/alpha-proof.ts`.

The suite covers CLI/help output, compiler Plan and source-map artifacts,
templates, example classification and PR smoke examples, diagnostics, package
manifests, and alpha app flows. Goldens are updated only through the explicit
update wrappers:

```powershell
tools/windows/update-goldens.ps1 -Area all
tools/windows/update-goldens.ps1 -Area cli -Verify
```

```sh
tools/unix/update-goldens.sh --area all
tools/unix/update-goldens.sh --area cli --verify
```

The harness normalizes checkout paths, temp roots, line endings, local ports,
timestamps, and durations. It does not normalize route order, Plan semantics,
diagnostic wording, package manifest shape, source-map metadata, or generated
artifact presence.
