Sloppy Audit

[note] SLOPPY_AUDIT_OS_POLICY_VISIBLE OS capabilities are policy-visible for sloppy/os; no OS sandbox, environment values, native handles, or pid-as-handle capabilities are exposed (capabilities)
[note] SLOPPY_AUDIT_OS_ENV_REDACTED environment metadata names access only and omits variable values (capabilities)
[note] SLOPPY_AUDIT_OS_PROCESS_EXPLICIT_ARGV process metadata includes current-process snapshots and explicit argv only (capabilities)
[warn] SLOPPY_AUDIT_OS_SHELL_DENIED shell capability metadata is visible, but shell execution is denied unless a future explicit gate enables it (capabilities)
[note] SLOPPY_AUDIT_OS_PLAN_VISIBLE sloppy/os is Plan-visible as stdlib.os (requiredFeatures)
